# Comment lancer le projet ?
1. Assurez-vous d'avoir Python installé sur votre machine.
2. Clonez le dépôt du projet ou téléchargez les fichiers nécessaires.
3. Installez les dépendances nécessaires en exécutant `pip install` ou alors utilisez des environnements virtuels comme venv ou conda pour gérer les dépendances de votre projet.
4. Lancez l'application en exécutant `python app.py`.

# Comment est structuré le projet ?
Le projet est structuré de la manière suivante :
- `app.py` : Le point d'entrée de l'application Dash.
- `pages/` : Un dossier contenant les différentes pages de l'application.
- `expli1.md`, `expli2.md`, `expli3.md` : Des fichiers Markdown contenant les explications pour chaque page.
