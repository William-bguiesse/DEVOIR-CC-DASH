# Accueil 

## Présentation du projet.
Ce projet a pour objectif de présenter les donnees de ventes d'avocat dans le monde. Nous pouvons constater 
dans les graphiques il est possible de trier les donnees par region des Etats-Unis.
Pour la seconde page, on peut retrouver notre tableau de donnees, avec la possibilite de trier les donnees selon le type d'avocat, la region ou encore le prix de vente.
