# Les Callback

Dans le projet, les callbacks sont utilisés pour rendre l'application interactive. Ils permettent de mettre à jour les graphiques et les tableaux en fonction des sélections de l'utilisateur. Pour tous les graphiques ou encore pour le tableaux de donnees, nous avons utilisé des callbacks pour que les utilisateurs puissent trier les données par région, type d'avocat ou prix de vente.
